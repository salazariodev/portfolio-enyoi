## Installation

**1 - Clone this repo:**
```
git clone https://github.com/salazariodev/portfolio-enyoi.git
```

**2 - Go to project dir:**
```
cd portfolio-enyoi
```
**3 - Remove the `.git` folder, so you can add it to your own repo later:**

**Linux/MacOS**
```
rm -rf .git
```
**Windows**
Using PowerShell
```
Remove-Item .git -Recurse -Force
```
